# مساعد التسوق الذكي — نسخة JSON

هذه النسخة تفصل البيانات عن منطق التطبيق:
- `offers.json` يحتوي على بيانات العروض
- `coupons.json` يحتوي على بيانات القسائم
- `script.js` يجلب البيانات باستخدام `fetch()`

## لماذا هذه البنية أفضل؟
- أسهل في التوسّع والصيانة
- أقرب لمشاريع Frontend الحقيقية
- يسهل لاحقًا استبدال JSON بـ API

## الملفات
- `index.html`
- `style.css`
- `script.js`
- `offers.json`
- `coupons.json`

## التشغيل المحلي
**مهم:** لأن `script.js` يستخدم `fetch()` لتحميل ملفات JSON، قد لا تعمل البيانات إذا فتحت `index.html` مباشرة عبر `file://` في بعض المتصفحات.

استخدم أحد الخيارات التالية:
1. VS Code + Live Server
2. Python:
   ```bash
   python -m http.server 8000
   ```
   ثم افتح:
   `http://localhost:8000`
3. ارفع المشروع إلى GitHub Pages أو Netlify أو Vercel

## التطوير لاحقًا
- نقل JSON إلى مجلد `data/`
- إضافة صور للمنتجات
- استبدال JSON بواجهة API
- إضافة صفحة إدارة لإدخال العروض والقسائم
